InfluxDB.setPrecision public
Description
Especifica la precisión en la que InfluxDB devolverá el timestamp en los queries. Por defecto, InfluxDB devolverá el timestamp en nanosegundos. Para especificar una precisión diferente consultar la documentación de InfluxDB: InfluxDB - Other options when querying data
Params
Stringprecision
La precisión que se desea utilizar en los queries
Properties
Returns
  Notice
  Warning
Examples
Ej. Especificar precisión en segundos: 
$influx = new InfluxDB("estaciones", "umidev", "desarrollo123", "localhost");
            $influx->setPrecision("s");
