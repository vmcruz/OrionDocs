Documentation
InfluxDB.__constructor
Description
Permite crear una nueva instancia de InfluxDB
Params
Stringdb
Base de datos a la que se desea conectar de InfluxDB
Stringuser
Usuario de la base de datos
Stringpassword
Contraseña del usuario de la base de datos
Stringhost
Servidor donde se encuentra InfluxDB. Opcional, por defecto "localhost"
Stringport
Puerto de la API HTTP de InfluxDB. Opcional, por defecto "8086"
Properties
Returns
  Notice
  Warning
Examples
Ej. Nueva instancia de InfluxDB: 
$influx = new InfluxDB("umidb", "umidev", "desarrollo123", "192.168.1.99");
